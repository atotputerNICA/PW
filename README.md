# PW
The Secret Garden - Flower Shop

### Prerequisites
* [Heroku](https://signup.heroku.com/signup/dc) account and installed locally.
* [Php](http://php.net/) installed locally.
* [Composer](https://getcomposer.org/doc/00-intro.md) installed locally.

### Set up
```bash
> heroku login
> php -v
> composer -V
> git --version
```
